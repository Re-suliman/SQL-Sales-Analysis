# SQL Sales Analysis

## Overview
This project analyzes a retail sales dataset using SQL to uncover business insights from over 9,900 sales records.

## Dataset
- Sample Superstore Dataset
- ~9,994 sales records

## Tools
- SQL
- SQLite Online

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT, AVG)
- LIMIT

## Analysis Performed
- Total Sales
- Total Profit
- Total Orders
- Top Products
- Sales by Category
- Top Cities
- Top Customers
- Profit by Region
- Top States
- Top Sub-Categories

## Key Insights
- Technology generated the highest sales.
- The West region achieved the highest overall profit.
- A small number of products contributed significantly to total revenue.
